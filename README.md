# Chef到家 - 厨师上门服务平台

> 连接用户与厨师的到家做饭平台

## 📱 项目简介

用户可以浏览厨师、菜品，下单让厨师上门做饭；厨师可以入驻接单。

## ✨ 核心功能

- 用户端：浏览厨师/菜品、下单、购物车、订单管理、个人中心
- 厨师端：入驻申请、接单、菜品管理、收入管理

## 🛠️ 技术栈

- 前端：Vite + Vue 3
- 后端：FastAPI (Python)
- 数据库：PostgreSQL / MySQL
- 缓存：Redis
